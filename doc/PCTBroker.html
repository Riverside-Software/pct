PCTBroker task

Description

Manages and maintains property files.

This task is able to create and delete databases, appservers brokers and webspeed brokers entries. This task is using the MergeProperties class.

This task cannot be run by itself. You need to run one of its subclasses.

Parameters

Attribute Description Type Requirement Default value
action Action to execute. Should be "create|delete|update" String Required No default value
name Broker name to update/delete String Required No default value
file Properties file to update File Optional $DLC/properties/[conmgr|ubroker].properties
UID Generate unique ID ? String None, auto or given UID None
