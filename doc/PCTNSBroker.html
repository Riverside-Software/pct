PCTNSBroker task

Description

Adds, updates and deletes Name Server broker declarations

Parameters

Attribute Description Type Requirement Default value
Name NameServer's name String Required NS1
Action Must be one of create, update, delete String Required No default value
WorkDir Working directory File Optional No default value
PortNumber Integer Optional No default value
AutoStart Boolean Optional False
ServerLogFile Server's log file File Optional No default value
ServerLogLevel Should be between 1 and 5 Integer Optional No default value
LogAppend Append to log file Boolean Optional True
brokerKeepAliveTimeout Keep Alive Timeout Integer Optional No default value
Name Environment Name, usually set to NS1 String Required No default

Definition of servers processes ; Inherits from attributes of PCTRun task

Examples

<PCTNSBroker action="create" name="NS1" dlcHome="${env.DLC}" />

Simplest declaration : this will create a [NameServer.NS1] section in the $DLC/properties/ubroker.properties file. In this case, every attribute is inherited from parent sections. 

<PCTNSBroker action="update" name="NS1" dlcHome="${env.DLC}" portNumber="1234" />

Updates the previously created Name Server, by adding portNumber property. 

<PCTNSBroker action="delete" name="NS1" dlcHome="${env.DLC}" />

Deletes the previously created Name Server. 

<PCTNSBroker action="update" nameServer="NS1" name="NS1" dlcHome="${env.DLC}" file="myubroker.properties">
  <portNumber="2000" autoStart="true" serverLogFile="NS1.ns.log" >
  <brokerKeepAliveTimeout="30" >
</PCTNSBroker>

Nameserver's definition will look like this :

[NameServer.NS1]
    autoStart=1
    brokerKeepAliveTimeout=30
    environment=NS1
    logAppend=1
    portNumber=2000
    srvrLogFile=NS1.ns.log

This definition will be integrated in myubroker.properties instead of standard $DLC/properties/ubroker.properties
