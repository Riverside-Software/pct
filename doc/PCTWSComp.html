PCTWSComp task

Description

Converts Webspeed HTML files to .w

Parameters

Attribute Description Type Requirement Default value
destDir Directory where to put .w files File Required No default value
failOnError If task should end after a failed file transformation Boolean Optional False
forceCompile Always compile everything Boolean Optional False
Debug Adds debug to e4gl-gen.p options Boolean Optional No
KeepMetaContentType Adds keep-meta-content-type to e4gl-gen.p options Boolean No No
WebObject Adds web-object to e4gl-gen.p options. If false, adds include option. Boolean No True

PCTWSComp inherits attributes from PCT and PCTRun.

Parameters as nested elements

fileset (org.apache.tools.ant.types.Fileset)

Adds a file set to the file list to convert

Examples

<PCTWSComp destDir="build" dlcHome="${env.DLC}">
  <fileset dir=".">
    <include name="test.html"/>
  </fileset>
</PCTWSComp>

Converts test.html and put test.w in build directory.
